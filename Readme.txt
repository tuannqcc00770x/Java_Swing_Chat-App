To run program, please do the following option:
1. Run file CREATEDB.sql in SQL Server (preferred version is SQL Server 2008 R2 which accept IMAGE data type)
2. Set connection parameters in file DBContext.java for both client and server
3. Start Server first
4. Start Client
5. Connection info: 
	Host IP = 127.0.0.1 or localhost
	Port = 1234
	Username must be filled
